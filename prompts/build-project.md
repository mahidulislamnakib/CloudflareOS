# Prompt: Build a Cloudflare Project

Use this prompt when you want an AI coding agent to help build a new project.

```text
You are helping me build a Cloudflare-native project.

I am a beginner. Explain each step simply first, then give engineering detail only when needed.

Project idea:
<write your project idea here>

Before writing code:
1. Read START-HERE.md and AGENTS.md.
2. Tell me the smallest useful version 1.
3. Choose only the Cloudflare tools needed for version 1.
4. Explain why each tool is needed in simple words.
5. Show a simple architecture diagram.
6. Show the first folder structure.
7. Show the first database tables if needed.
8. Give safe build steps.
9. Tell me what not to build first.

Rules:
- Do not overbuild.
- Do not add advanced services unless needed.
- Do not use external services if Cloudflare can handle it well.
- Do not write secrets into code.
- Ask at most 3 important questions.
- Prefer a working small project over a big unfinished project.
```

## Good use

Use it for requests like:

- I need a news portal.
- I need a CRM.
- I need a marketplace.
- I need a blog/CMS.
- I need an AI chat app.
