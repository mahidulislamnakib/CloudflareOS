# Start Here

Welcome. This project helps you build websites and apps using Cloudflare.

You do **not** need to be an expert.

If you can use a computer, read basic English, and copy commands carefully, this guide is for you.

## What is Cloudflare Engineering OS?

It is a guide and toolkit that helps you answer:

- What should I build?
- Which Cloudflare tools should I use?
- Where should I put my code?
- How do I deploy?
- How do I fix errors?
- How do I keep the project safe?

## Simple idea

Think of Cloudflare like a toolbox.

| Need | Simple Cloudflare tool |
| --- | --- |
| Website page | Pages |
| App/API code | Workers |
| Database | D1 |
| File upload | R2 |
| Fast small cache | KV |
| Background task | Queues |
| Step-by-step process | Workflows |
| AI feature | Workers AI |
| Protect a form | Turnstile |
| Protect admin area | Access |

## Beginner path

Follow this order:

1. Read this file.
2. Read `docs/02-newcomer-roadmap.md`.
3. Pick one small project.
4. Build only the simple version first.
5. Deploy.
6. Add advanced features later.

## Best first project

Build a small blog or CMS:

- Show posts on a website.
- Save posts in D1.
- Upload images to R2.
- Protect admin with Access or login.
- Deploy with Wrangler.

Do not start with a huge ERP, marketplace, or social network. Learn the flow first.
