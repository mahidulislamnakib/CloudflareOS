# Cloudflare Engineering OS

> **Build Cloudflare-native applications like a senior platform engineer.**

Cloudflare Engineering OS is an AI-ready engineering operating system for planning, architecting, building, debugging, deploying, securing, monitoring, and continuously improving Cloudflare-native applications.

---

## Vision

This repository is **not** a starter template.

It is a complete engineering system that combines:

- Cloudflare product knowledge
- AI coding-agent rules
- Reference architectures
- Production checklists
- Debug playbooks
- Deployment guides
- Update automation

The goal is simple:

> From idea → production, using Cloudflare-first engineering.
